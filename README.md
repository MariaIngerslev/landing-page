---

## Features

### Navigation
- Logo on the left and navigation links on the right  
- Fully styled using Flexbox  
- Cursor pointer added to improve UX  

### Hero Section
- Large headline, descriptive subtext, and a call-to-action button  
- Styled image placeholder with fixed dimensions  

### Information Section
- Four information cards  
- Each card includes a reusable placeholder component  
- Layout created with Flexbox  

### Quote Section
- Soft gray background  
- Large italic quote text  
- Author aligned to the right  

### Sign-Up Section
- Blue call-to-action container (color: `#2563EB`)  
- Matching button color for visual consistency  
- Includes a title, subtext, and CTA button  

### Footer
- Simple centered footer  
- Dark background with white text
